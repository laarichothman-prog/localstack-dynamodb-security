# LocalStack - Sécurisation DynamoDB locale

## Description
Mini projet Cloud Computing & Virtualisation utilisant LocalStack pour simuler AWS DynamoDB localement avec des mécanismes de sécurité :
- contrôle d’accès,
- chiffrement,
- audit,
- tests de sécurité.

## Technologies
- Docker
- LocalStack
- DynamoDB
- Python
- boto3

## Lancement

```bash
docker compose up -d
python demo_complete.py
